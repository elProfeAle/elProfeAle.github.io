Ejemplo de Css con flexbox

Podés copiar el contenido del ejemplo y pegarlo en tu archivo .css para probar, retomamos la página utilizada en el ejemplo de html:

css 
body{
    font-family: Arial, Helvetica, sans-serif;
    background-color: rgb(236, 236, 236);
}

/*Selecciono el id tituloPrincipal*/
#tituloPrincipal{
    color: rgb(3, 120, 216);
}

/*Selecciono la clase presentacion*/
.presentacion{
    /*Cambia la alineación del texto*/
    /*En este caso, lo centra*/
    text-align: center;
}

/*Selecciono la clase servicios*/
.servicios{
    /*Utilizo flex en una caja (DIV)*/
    display: flex;
    justify-content: space-around;
}

/*Selecciono la clase tarjeta*/
.tarjeta{
    width: 40%;
    /*Utilizo flex en una caja (DIV)*/
    display: flex;
    flex-direction: column;
    align-items: center;
    /*Agregamos un borde a la caja*/
    border: 1px solid grey;
    /*El borde lo redondeamos*/
    border-radius: 8px;
    /*Damos espaciado entre el borde y el contenido*/
    padding: 10px;
    background-color: white;
}

/*Selecciono solo los parrafos que corresponden a la clase tarjeta*/
.tarjeta p{
    /*Justificamos el texto*/
    text-align: justify;
}
