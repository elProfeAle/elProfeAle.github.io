Selectores en CSS

Los selectores en CSS son patrones utilizados para seleccionar los elementos que se desean estilizar.

En esta guía, aprenderás sobre los diferentes tipos de selectores y cómo utilizarlos para aplicar estilos a tus páginas web.

Tipos de selectores

1. Selector de tipo

Selecciona todos los elementos de un tipo determinado (etiqueta HTML).

h1 {
  font-size: 2em;
}

2. Selector de clase

Selecciona elementos con una clase específica. Se usa un punto (.) seguido del nombre de la clase. 

.destacado {
  background-color: yellow;
}

3. Selector de ID

Selecciona un elemento por su ID. Se escribe con una almohadilla (#).

#tituloPrincipal {
  text-align: center;
}

4. Selector universal

Selecciona todos los elementos del documento.

* {
  margin: 0;
  padding: 0;
}

5. Selector de descendencia

Selecciona elementos que están dentro de otros elementos.

article p {
  line-height: 1.5;
}

6. Selector de grupo

Permite aplicar los mismos estilos a varios elementos a la vez.

h1, h2, h3 {
  font-family: Arial, sans-serif;
}
