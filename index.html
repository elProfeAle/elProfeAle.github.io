Programación II

Bienvenidos al sitio web de Programación II del Instituto Visión Tecnológica, el siguiente sitio web contiene información sobre diferentes conceptos de diseño web vistos en Programación II. Principalmente van a encontrar información sobre los 3 lenguajes que se utilzarán:

HTML es el lenguaje que usamos para crear la estructura de una página web. Es como el esqueleto de un sitio: define los elementos básicos como títulos, párrafos, imágenes, enlaces y más. Con HTML organizamos el contenido para que el navegador lo entienda y lo muestre.

CSS es el lenguaje que usamos para darle estilo y diseño a una página web. Es como la ropa del sitio: cambia colores, tamaños, posiciones y más. Con CSS podés hacer que un título sea rojo, centrar un párrafo, o alinear cajas con Flexbox.

JavaScript es el lenguaje que usamos para agregar interactividad a una página web. Es como el cerebro del sitio: con JS podés crear botones que muestren mensajes, formularios que se validen, o juegos simples.
